>[力扣中国 面试题 08.11. 硬币](https://leetcode-cn.com/problems/coin-lcci/)
题目：给定数量不限的硬币，币值为25分、10分、5分和1分，编写代码计算n分有几种表示法。(结果可能会很大，你需要将结果模上1000000007)

<!--more-->
#### 思路
一条非常好的题，从问题的思考，到一步一步的优化，对深入理解动态规划、背包问题以及滚动数组有非常大的帮助。思路参照乐扣中国给出的官方分析，非常到位，这里不做重复描述：[官方解答](https://leetcode-cn.com/problems/coin-lcci/solution/ying-bi-by-leetcode-solution/)

这道题本质是完全背包问题，在限定的体积V（题中的n），达到V的最大组合方案数

#### 算法实现
```java
public int waysToChange(int n) {
        int[] dp = new int[n + 1];
        int[] coins = {1, 5, 10, 25};
        dp[0] = 1;
        for (int i = 0; i < coins.length; i++) {
            int coin = coins[i];
            for (int j = coin; j <= n; j++) {
                dp[j] = dp[j] + dp[j - coin];
            }
        }
        return dp[n];
    }
```
